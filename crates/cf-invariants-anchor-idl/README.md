# cf-invariants-anchor-idl

Anchor IDL ingestion for the
[cf-invariants-anchor](https://github.com/caliperforge/cf-invariants-anchor)
workspace: parses Anchor 0.30 / Codama-style IDL JSON into the typed
contract surface defined in `cf-invariants-anchor-core`, which the
suggester and emitters consume downstream.

MSRV is pinned at Rust 1.79 via the workspace `rust-version`.
