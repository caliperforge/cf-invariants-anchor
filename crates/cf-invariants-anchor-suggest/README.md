# cf-invariants-anchor-suggest

Ranked invariant suggestion for the
[cf-invariants-anchor](https://github.com/caliperforge/cf-invariants-anchor)
workspace: walks a parsed Anchor contract surface and proposes candidate
invariants from the class library — Phase 0 ships
`balance_conservation`, `monotonic_accounting`, and `access_control`.
The deterministic heuristic path lives here; the AI path lives in
`cf-invariants-anchor-ai`.

MSRV is pinned at Rust 1.79 via the workspace `rust-version`.
