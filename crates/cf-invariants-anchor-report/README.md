# cf-invariants-anchor-report

Scorecard rendering for the
[cf-invariants-anchor](https://github.com/caliperforge/cf-invariants-anchor)
workspace: renders harness-run results as markdown and JSON scorecards,
mirroring the format of the Cairo-targeted sibling project
`cf-invariants` (Starknet / snforge).

MSRV is pinned at Rust 1.79 via the workspace `rust-version`.
